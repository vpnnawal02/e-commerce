# MERN E-Commerce Website
A full-stack E-Commerce web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).
This project is currently under active development and will soon support full shopping functionality including authentication, product management, cart, and payments.

# Project Status
Under Production
Core architecture setup in progress. Features are being developed incrementally.
